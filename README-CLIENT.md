# Deng Details Cleaning Services - Site Handoff

Welcome to your new website. This README is your owner's manual: what's where, what you can change, and what to ask Pejji for when you need updates.

## What's on the site

- **Home page** with hero, services, why-us, process, testimonials, contact
- **Privacy policy** page (PIPEDA-compliant by default)
- **404 page** that matches the brand
- **Mobile-first design** that looks great on phones
- **WhatsApp button** on every section so customers can book in one tap

- **Service descriptions** (currently 6 services - add or remove anytime)
- **Testimonials** (currently 3 anonymous quotes - swap in real client quotes when you have them)
- **Hero trust signals** ("Same-day quote / Pro crew / Pay after you approve") - adjust language anytime
- **Phone / WhatsApp number** (live)
- **Tagline** ("We don't just clean. We pay attention.") - swap if you want a new one
- **Color accent** (currently teal-blue from your logo) - we can shift to any other accent
- Add **photos** of your team, your equipment, before/after shots
- Add a **services pricing page** if you want clients to see prices before booking
- Add a **"book online" form** that creates a calendar event automatically (Pro tier)

5. **Real testimonials**: once you have happy clients, swap the placeholder quotes for real ones.
