# DDCS Cleaning - Vision Demo

This is a preview website built by Pejji for **Dang Detail Cleaning Services**. It's a vision demo showing how your business could be presented online. Take a look, let us know what you'd like to change, and we'll iterate based on your feedback.

## What's in here

- **Home page** with hero, services, why-us, process, testimonials, contact
- **Privacy policy** page (NDPA-compliant by default)
- **404 page** that matches the brand
- **Mobile-first design** that looks great on phones
- **WhatsApp button** on every section so customers can book in one tap
- **Cookie consent banner** that respects your customers' privacy
- **Security headers** built in from day 1 (HSTS, CSP, X-Frame-Options, etc)

## Things you can change easily

- All the **service descriptions** (currently 6 services - we can add or remove)
- All the **testimonials** (currently 3 placeholder ones with realistic-sounding names; replace with real customer quotes when you have them)
- The **trust numbers** in the hero (500+ cleans / 4.9 rating / 100% guarantee). If your real numbers are different, we update them.
- **Phone number / WhatsApp number** (currently placeholder - we use Pejji's number until you give us yours)
- **Tagline** in the hero ("We don't just clean. We pay attention.") - if you have a different one, we swap it in
- **Color accent** (currently teal-blue from your logo) - we can shift to any other accent
- Add **photos** of your team, your equipment, before/after shots, etc.
- Add a **services pricing page** if you'd like clients to see prices before booking
- Add a **"book online" form** that creates a calendar event automatically (Pro tier)

## Things to discuss

1. **Country / market**: this demo defaults to Nigerian market (₦, NDPA compliance, +234 phone). If you serve another country or both, tell us and we'll adjust pricing display, privacy policy, and contact details.

2. **Service area**: do you serve specific cities only? We can add a service-area map or list.

3. **Booking flow**: do you want customers to book entirely on WhatsApp, or should there be an "instant quote" form that calculates price based on home size?

4. **Recurring service plan**: if you offer subscriptions (weekly/bi-weekly), we can add a dedicated page that shows the savings.

5. **Photos**: a real photo of you and your team would replace the cartoon mascot in the hero. People trust faces.

## Tech notes (for the curious)

- Static HTML/CSS/JS - super fast loading, no database needed
- Hosted on Cloudflare Pages with global CDN
- All security headers enabled by default
- Built with web standards - no fragile JavaScript framework dependency
- Mobile load time under 2 seconds on 4G connection
